Documentation


Documentation is made so it can simplify your learning and make it easy.
We are making the documentation as it will not also make the users happier
but it will make a huge difference to the people that are using SophusGUI.


First, you have to create a python file any name, I called it main.py . After,
just type import SophusGUI . Follow thee tutorial to make other code under here.
Now to create a gui app write create_gui_app() . Click Run and then you will
have a 900x500, lightgray background GUI application. If you want a button
just type gui_button() . There are a lot of buttons to see but you have to
view it in the code file. To make a different background colour just type
guiBg_Red() to make the red background colour and there also many different
background colours just view them in the code file. There are many built-in
titles like Application, SophusGUI and many to make them type this :
guiTitle_SophusGUI() or guiTitle_Application() and there many in the code
file. Next you have the CheckBoxes or RadioButtons to make them type this:
guiCheckBox_Countries and again there are many samples of these.
